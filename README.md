# BusCrafter

## Wymagania systemowe:
- System operacyjny: Windows 10/11 (7 lub 8.1 mogą działać, ale nie były testowane)
- Dostępna przestrzeń dyskowa: ∼60 MB
- Pamięć RAM: minimum 2 GB

## Program:
Program pisany w Pythonie służy do tworzenia różnych rzeczy związanych z autobusami/tramwajami w Krakowie. Głównym założeniem było tworzenie rozkładów jazdy, ale w przyszłości będzie rozbudowana o inne funkcje.

## Checklist funkcji:
- Rozkłady jazdy - schemat od 2022 roku ✅
- Rozkłady jazdy - schemat do 2022 roku ✅
- Tabliczki z nazwą przystanku ✅
- Tzw. "liniówki" - tabliczki z numerem linii na przystanku ✅
- Wyświetlacze ❌

  ❌ = jeszcze nie wprowadzona funkcja;
  ✅ = już wprowadzona funkcja
